# encrypt-decrypt-pyqt5

可以使用常见加密算法对文件或字符串进行加密和解密，GUI使用PyQt5.11.2编写。

## 实现以下古典密码对字符串的加密和解密：
1. 单表替代密码：
* Caesar cipher
* Keyword cipher
* Affine cipher
* Multilateral cipher

2. 多表替代密码：
* Vigenere cipher
* Autokey ciphertext
* Autokey plaintext

3. 多图替代密码：
* Playfair cipher

4. 置换密码：
* Permutation cipher
* Column permutation cipher
* Double-Transposition cipher
---
## 实现以下流密码对字符串和文件的加密和解密：
* RC4
* CA

## 实现以下分组密码对字符串和文件的加密和解密：

## 实现以下公钥密码密钥对的生成及对字符串和文件的加密和解密：
